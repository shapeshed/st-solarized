# st-solarized

An ArchLinux PKGBUILD for [st][1] with patches applied for the [Solarized][2] theme.

## Installation

    git clone git@github.com:shapeshed/st-solarized.git
    cd st-solarized
    makepkg -sri

## Usage

Refer to the [Arch Linux Wiki][3] for usage.
